# REDESIGN-A-WEBSITE-LANDING-PAGE

## Project Overview

This project focuses on redesigning an existing website landing page to improve usability, visual hierarchy, accessibility, and overall user experience. The redesign aims to create a more modern and engaging interface while maintaining the original purpose of the website.

## Objective

The primary objectives of this redesign are:

* Improve visual appeal and aesthetics
* Enhance user experience and usability
* Establish a clear visual hierarchy
* Improve readability and accessibility
* Increase user engagement and conversion potential

## Deliverables

### Files Included

* `before_design.png` – Screenshot of the original landing page.
* `after_design.png` – Redesigned landing page concept.
* `design_explanation.md` – Explanation of design improvements and decisions.

## Design Improvements

### Visual Hierarchy

Improved content organization through better typography, spacing, and section alignment.

### Navigation

Simplified navigation structure for easier access to important information.

### Typography

Enhanced readability through improved font sizing, spacing, and content structure.

### Color Scheme

Applied a modern and consistent color palette to improve aesthetics and brand perception.

### Call-to-Action (CTA)

Made primary actions more visible and accessible to encourage user interaction.

### Responsive Design Considerations

Ensured the redesigned layout remains usable across different screen sizes.

## Tools Used

* Figma
* UI/UX Design Principles

## Conclusion

The redesigned landing page provides a cleaner interface, improved usability, and a more engaging user experience while maintaining the original functionality and content objectives.
